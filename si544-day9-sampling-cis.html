Sampling distributions

# Segment 1: sample size = 25 ------------------------------
# 1.a) Virtually use shovel 1000 times
virtual_samples_25 <- bowl %>% 
  rep_sample_n(size = 25, reps = 1000)

# 1.b) Compute resulting 1000 replicates of proportion red
virtual_prop_red_25 <- virtual_samples_25 %>% 
  group_by(replicate) %>% 
  summarize(red = sum(color == "red")) %>% 
  mutate(prop_red = red / 25)

# 1.c) Plot distribution via a histogram
ggplot(virtual_prop_red_25, aes(x = prop_red)) +

jamie_candy <- read_csv("candy_alex.csv")
## Parsed with column specification:

jamie_candy <- jamie_candy %>% mutate(nut_status = case_when(Name %in% c("M&M Plain","M&M Peanut","Snickers","Twix") ~ "nut",TRUE ~ "nutfree"))

jamie_candy %>% specify(response = nut_status, success = "nutfree") %>% calculate(stat="prop")
## # A tibble: 1 x 1
##    stat
##   <dbl>
## 1   0.8
jamie_candy_bootstrap <- jamie_candy %>%
  specify(response = nut_status, success = "nutfree") %>%
  generate(reps = 1000, type = "bootstrap") %>%
  calculate(stat = "prop")

percentile_ci_candy_jamie <- jamie_candy_bootstrap %>%
  get_confidence_interval(level = 0.95, type = "percentile")
percentile_ci_candy_jamie
## # A tibble: 1 x 2
##   `2.5%` `97.5%`
##    <dbl>   <dbl>
## 1    0.6    0.95
jamie_candy_bootstrap %>% visualize(bins=10) +
  shade_confidence_interval(endpoints=percentile_ci_candy)

Comparing Jamie’s sample to Alex’s sample, it does not appear that there is a significant bias towards nut or nutfree candy. Both samples contain the estimated true proportion of nutfree candy of 0.8 or 80% with the 95% confidence interval. In both visualizations, we can see that the samples cluster around 0.8 with only a few falling outside of the confidence interval range. Jamie’s distribution of samples appears to be higher than 0.8 whereas Alex’s samples are slightly below. However, this could be a product of sampling variation and not necessarily an indication of bias.
