Bootstrapping the slope

bootstrap_distn_slope <- evals_ch6 %>% 
  specify(formula = score ~ bty_avg) %>%
  generate(reps = 1000, type = "bootstrap") %>% 
  calculate(stat = "slope")
visualize(bootstrap_distn_slope)

Hypothesis tests around the slope

null_distn_slope <- evals %>% 
  specify(score ~ bty_avg) %>%
  hypothesize(null = "independence") %>% 
  generate(reps = 1000, type = "permute") %>% 
  calculate(stat = "slope")
visualize(null_distn_slope)

visualize(null_distn_slope) + 
  shade_p_value(obs_stat = observed_slope, direction = "both")

Learning Check 11.2 (10 minutes)

Repeat the inference above but this time for the correlation coefficient instead of the slope. Note the implementation of stat = “correlation” in the calculate() function of the infer package.

correlation_null <- evals %>%
  specify(score ~ bty_avg) %>%
  hypothesize(null = "independence") %>% 
  generate(reps = 1000, type = "permute") %>% 
  calculate(stat = "correlation")

visualize(correlation_null)

observed_corr <- evals %>%
  specify(score ~ bty_avg) %>%
  calculate(stat = "correlation")

observed_corr
## # A tibble: 1 x 1
##    stat
##   <dbl>
## 1 0.187
correlation_null %>%
  get_p_value(obs_stat = observed_corr, direction = "both")
## # A tibble: 1 x 1
##   p_value
##     <dbl>
## 1       0
visualize(correlation_null) + 
  shade_p_value(obs_stat = observed_corr, direction = "both")

Given an alpha of 0.05 and a p value as extreme as the one we see, we have sufficient evidence to reject the null hypothesis that the correlation coefficient between score and beauty average is 0.
